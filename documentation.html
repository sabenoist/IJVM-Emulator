CS045 Hakuna Matata Web API

GET /table

Returns the whole table from the database as a JSON object.

Resource Information

Endpoint http://127.0.0.1:8080/table
Request Method GET
Response Format JSON

Example Request

GET http://127.0.0.1:8080/table

Example Result

[{“id”:1, “name”:”Anakin Skywalker”, “category”:”Jedi Padawan”, “location”:”Coruscant”, “date”:”2002-05-16”, “amount”:1}, {“id”:2, “name”:”Han Solo”, “category”:”Space Badass”, “location”:”The Millennium Falcon”, “date”:”1977-12-15”, “amount”:1}, {“id”:3, “name”:”Yoda”, “category”:”Jedi Master”, “location”:”Coruscant”, “date”:”2002-05-16”, “amount”:1}]

Possible Status Codes

200 The server returns the requested table.

GET /table/id

Returns a row from the table based on the id that is given as a parameter.

Resource Information

Endpoint http://127.0.0.1:8080/table/id
Request Method GET
Response Format JSON

Example Request

GET http://127.0.0.1:8080/table/1

Example Result

[{“id”:1, “name”:”Anakin Skywalker”, “category”:”Jedi Padawan”, “location”:”Coruscant”, “date”:”2002-05-16”, “amount”:1}]

Possible Status Codes

200 The server returns the requested table.
204 The requested item has been deleted from the database.
404 The requested item could not be found in the database.

POST /table

Adds a row to the table in the database based on the information that accompanies the request as a JSON object.

Resource Information

Endpoint http://127.0.0.1:8080/table
Request Method POST
Response Format JSON

Example Request

POST http://127.0.0.1:8080/table

Example Request Body

[{“name”:”Mace Windu”, “category”:”Jedi Master”, “location”:”Coruscant”, “date”:”2002-05-16”, “amount”:1}]

Example Response

{"ITEM id 1 ADDED": "http://localhost:8080/table/1"}

Possible Status Codes

201 The item has been added to the table.
405 The JSON format is incorrect or the object is missing.
507 The server could not process the request because the database has insufficient space.

PUT /table/id

Edits a row from the table based on the id as a parameter and the JSON object that accompanies it.

Resource Information

Endpoint http://127.0.0.1:8080/table/id
Request Method PUT
Response Format JSON

Example Request

PUT http://127.0.0.1:8080/table/1

Example Request Body

[{“name”:”Darth Vader”, “category”:”Sith Lord”, “location”:”Coruscant”, “date”:”2002-05-16”, “amount”:1}]

Possible Status Codes

204 The requested item has been deleted from the database and cannot be edited.
404 The requested item to edit could not be found in the database.
507 The server could not process the request because the database has insufficient space.

DELETE /table/id

Deletes a row from the table in the database based on the id as a parameter.

Resource Information

Endpoint http://127.0.0.1:8080/table/id
Request Method DELETE
Response Format JSON

Example Request

DELETE http://127.0.0.1:8080/table/2

Possible Status Codes

204 The server has accepted the request and deleted the row.
404 The requested item to edit could not be found in the database.
